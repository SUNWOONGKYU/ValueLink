Gemini CLI 요청문 - 92개 기업의 투자 뉴스 URL 찾기
=================================================================

## 작업 목표
92개 한국 스타트업 기업의 투자 유치 뉴스 기사 URL을 찾아주세요.
이 기업들은 2026년 1월에 투자를 받은 기업들이며, 모든 기업에 대한 뉴스가 반드시 존재합니다.

## 입력 데이터
파일: not_found_group1.csv, not_found_group2.csv, not_found_group3.csv
- 총 92개 기업
- 각 기업에 대해: 기업명, 주요사업, 투자자, 투자단계, 투자금액 정보 포함

## 검색 대상 미디어 (우선순위 순)
1. 벤처스퀘어 (https://www.venturesquare.net/)
2. 스타트업투데이 (https://startuptoday.kr/)
3. 더브이씨 (https://thevc.kr/)
4. 비석세스 (https://besuccess.com/)
5. 아웃스탠딩 (https://outstanding.kr/)
6. 플래텀 (https://platum.kr/)
7. 블로터 (https://www.bloter.net/)
8. 스타트업엔 (https://startupn.kr/)
9. WOWTALE (https://www.wowtale.net/)
10. 넥스트유니콘 (https://www.nextunicorn.kr/)
11. 이코노미스트 (https://www.economist.co.kr/)

## 검색 방법
각 기업명에 대해:
1. "{기업명} 투자 유치" 키워드로 각 미디어 사이트 검색
2. 검색 결과에서 투자/유치/펀딩/시리즈 키워드가 포함된 기사 찾기
3. 해당 기사의 URL 추출

## 출력 형식
CSV 파일로 저장:
- 컬럼: 기업명, 주요사업, 투자자, 단계, 신규, 주차, 뉴스URL, 뉴스소스

예시:
```csv
기업명,주요사업,투자자,단계,신규,주차,뉴스URL,뉴스소스
이노션스,미용 의료기기 스타트업,알토스벤처스 등,시리즈A,142억원,5주차,https://www.venturesquare.net/...,벤처스퀘어
```

## 성공 기준
- 92개 기업 중 최소 80개 이상의 뉴스 URL 발견
- 각 URL은 실제 접속 가능한 기사 페이지여야 함
- 뉴스 소스명 정확히 기록

## 추가 정보
- 모든 기업은 2026년 1월에 투자를 받았으므로 최근 뉴스임
- 센서블박스 위클리에서 수집한 데이터이므로 뉴스가 반드시 존재함
- 일부 기업은 영문명 또는 약칭으로 보도되었을 수 있음

## 처리 시 주의사항
1. 각 사이트마다 검색 URL 형식이 다름 (위 URL 참고)
2. 일부 사이트는 로봇 차단이 있을 수 있으므로 적절한 헤더 사용
3. 요청 간 0.5초 대기로 사이트 부하 방지
4. 동일 기업이 여러 사이트에 보도된 경우 가장 상세한 기사 우선
